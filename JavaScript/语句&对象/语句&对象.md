## 复合语句
  + BlockStatement
```
  Block表达式语句与对象的表现方式相同，所以对象不允许在一个语句的开头; Block会把多条语句合成一条语句表示出来
  Block 形成新的作用域
  Block 在执行到 非normal 语句的时候就会终端 不再继续执行
  {
    [[type]]: normal
    [[value]]: --
    [[target]]: --
  }
```
